# Counsel — AI를 위한 심리 프레임워크

14개 근거 기반 심리상담 프레임워크, 하나의 진입점.
고민을 말하면 `/counsel`이 최적의 접근법을 자동 선택합니다.

> **이것은 치료가 아닙니다.** 심리 프레임워크를 배우고 자기 탐색에 활용하는 학습 도구입니다.

## `/counsel` — 하나의 명령어로 14개 프레임워크

```
/counsel 자꾸 같은 생각이 맴돌고 미래가 불안해서 잠을 못 자겠어
```

AI가 자동으로:
1. 위기 신호 스크리닝 (감지 시 즉시 안전 프로토콜)
2. 상황 분석 (감정 유형, 강도, 기간, 맥락)
3. 최적 프레임워크 1~3개 선택
4. 구조화된 자기 탐색 가이드 제공
5. 자기 연습 과제 제안

## 명령어

| 명령어 | 설명 |
|--------|------|
| `/counsel` | 전체 분석 → 프레임워크 선택 → 가이드 탐색 |
| `/counsel:select` | 빠른 프레임워크 추천만 |
| `/counsel:practice` | 구조화된 자기 탐색 세션 (5~10 교환) |

## 안전 장치

- **위기 감지**: 자살/자해 키워드 → 즉시 위기 프로토콜
- **위기 연락처**: 1393 (24시간), 1577-0199, 1588-9191
- **면책 조항**: 모든 세션 시작 시 "전문 심리치료를 대체하지 않습니다"
- **전문가 연계**: 전문 상담 권장 기준 명시

## 한국 문화 적응

- 집단주의 — 개인 욕구 탐색 시 죄책감 정상화
- 체면 — 감정을 "약함"이 아닌 "정보"로 재프레이밍
- 한 — 병리화하지 않고 문화적 맥락에서 이해
- 눈치 — 직접 말하지 않는 부분도 맥락 추론
- 수치심 문화 — "프레임워크 학습" 포지셔닝
